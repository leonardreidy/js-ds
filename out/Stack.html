Class: Stack

Stack

A Stack is essentially a list of data elements which may be accessed at only one end. Stacks are last-in, first-out (LIFO) data structures. A Stack ADT includes the following operations:
  • clear - to delete the contents of the Stack;
  • getLength - to get the number of elements contained in the Stack;
  • peek - to peek at the element at the top of the Stack;
  • pop - to remove elements from the top of the Stack;
  • push - to add elements to the top of Stack;
  • toString - to get a string representation of the Stack;

pop() → {*}

Pop element off the stack. This method is essentially the reverse of the push() function returning the element in the top position of the stack and decrementing the top variable. If a caller attempts to pop elements off an empty stack the stack is cleared (reset).
