Class: List

List

A List is an ordered sequence of data elements. New elements are appended to the end of a list with the append operation. Elements can be inserted into a list after an existing element or at the beginning of the list with the insert operation. Elements can be removed from a list with the remove operation. All elements in a list can be removed en masse with the clear operation. The toString operation returns a string representation of the list. The getElement returns the current element. The next operation is used to traverse the list from left to right, the prev element from right to left. The moveTo operation allows the caller to move to any arbitrary location in the list. The contains operation is used to see if a particular element can be found in the list. Lists have properties that include the front of the list and the back of the list. The isEmpty operation determines whether or not the list is empty. The currPos property stores the current position in the list. The length property stores the number of elements in a list.
